
# LyricGrabber

A web app to scrape and browse lyrics from Genius.com. Built with Vite + React.

## Features
- Scrape artist, album, and song lyrics from Genius.com
- Browse artists, albums, and songs in a modern UI
- View lyrics for each song

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- `src/` - React frontend code
- `.github/copilot-instructions.md` - Copilot custom instructions
